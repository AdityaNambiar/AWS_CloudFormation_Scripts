# AWS CloudFormation Scripts
Repository having AWS CloudFormation Scripts for study

The CloudFormation templates are defined in each folder based on the type of template
E.g. 
- instance : Consist of templates related to EC2, EBS, KeyPairs etc.
- storage  : Consist of templates related to S3

CloudFormation Stacks can be created either using traditional AWS CLI commands or with the helper script present in this repository named - "`run_aws_cfm.sh`"
